# barelytics.js

The tracking script for [Barelytics](barelytics.io).

### Development

The `dev` script will watch for changes and copy them to the web-app project.
The script assumes `barelytics.web-app` and `barelytics.js` are in the same parent directory.

```bash
npm install
npm run dev
```
